Push Order

This section covers the complete workflow for pushing orders through the warehouse system, including order pool management and courier selection.

Order Pool Process:

  1. Go to Outbound ‚Üí Order Pool
  2. Select orders that should be done first (priority orders)
  3. Select each courier
  4. Important: Ensure total number of orders does not exceed 10
  5. Create wave

Order Processing:

  1. Go to Outbound ‚Üí Order
  2. Single Same: Push the first 10 items
  3. Single Differ: Filter JNT
  4. Multi Differ: Filter JNT
  5. Multi Same: Process accordingly

After 1PM Processing:

  1. Go to Outbound ‚Üí Order
  2. Single Differ: Filter by shipping deadline (choose last 7 days)
  3. Multi Differ: Filter by shipping deadline (choose last 7 days)
  4. Apply store filter (copy store name and paste in appropriate box)
  5. Ensure total items do not exceed 20

Important Notes:

  • Before 1PM: Use JNT filter for Single Differ and Multi Differ
  • After 1PM: Use shipping deadline filter (last 7 days) instead of JNT filter
  • Single Differ: Ensure total items do not exceed 20
  • Multi Differ: Apply filter by store (copy store name and paste in appropriate box), ensure total items do not exceed 20
